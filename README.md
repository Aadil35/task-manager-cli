# task-manager-cli

**Assignment 3 — Aadil Jamali**

Java command-line task manager for the Git & GitHub feature branch workflow assignment.

## Requirements

- Java 17+

## Build and run

```bash
javac -d out src/taskmanager/*.java
java -cp out taskmanager.Main add "Buy groceries"
java -cp out taskmanager.Main show
java -cp out taskmanager.Main delete 1
```
